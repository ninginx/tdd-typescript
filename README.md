#### 俺の理解
12章では、Expression(式,要するに計算)という新しいクラスを実装する。Moneyクラスはあくまで、1ドルなどの単体のお金を表現するもので通過が違う場合には、Expression(式)を使って為替計算した上で合計金額を表すことになるらしい。例えば、1$ + 5 CHFが返すオブジェクトはExpressionになるらしい。

#### 振り返り
* 大きいテスト(10$ + 5CHF)を分割して、小さいテスト(5$+5$)を作成した
